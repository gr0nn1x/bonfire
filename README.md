## O aplikaci
Bonfire Fitness je multiplatformní aplikace (iOS, Android, Web) pro správu silových tréninků. Nabízí live tracking sérií, zápis vah, opakování a RPE v moderním dark mode designu.

## Tech Stack
* **Frontend:** React Native & Expo
* **Styling:** Tailwind CSS (NativeWind)
* **Backend:** Supabase (PostgreSQL)
* **Deployment:** Vercel

## Rychlý start
1. **Instalace:** `yarn install`
2. **Konfigurace:** Nastav `.env` (Supabase URL a Key)
3. **Spuštění:** `npx expo start -c`
